# **To-Do List Web Application**

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript. It allows users to add, edit, delete, and mark tasks as completed. The app supports multi-line tasks and stores all tasks using localStorage for persistence.

## Project Description

- The Smart To-Do List Web Application is designed to manage daily tasks efficiently.
- It supports adding, editing, deleting, and marking tasks as completed.
- Tasks are saved automatically using localStorage, ensuring that your list remains even after refreshing or closing the browser.
  
## Features

- Add new tasks easily
- Edit existing tasks with one click
- Delete tasks instantly
- Mark tasks as completed
- Multi-line task support for long descriptions
- Auto-save using localStorage (tasks remain after refresh)
- Responsive UI for smooth usage on all screen sizes

## Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling, layout, responsiveness
- **JavaScript (ES6)** – Functionality (add/edit/delete/complete tasks, localStorage)
- **LocalStorage** – Stores tasks permanently in the browser

## How to Install and Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Run index.html in any web browser.

### Optional (Using VS Code + Live Server)

1. Open the folder in VS Code.
2. Install the "Live Server" extension.
3. Right-click index.html → Open with Live Server.
   
## How to Use the Project

- Type a task in the input field.
- Click "Add Task" to add it to the list.
- Click the task text to mark it as completed.
- Click "Edit" to update a task.
- Click "Delete" to remove a task.
- All tasks are saved automatically using localStorage.
